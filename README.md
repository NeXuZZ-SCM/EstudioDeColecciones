# Snippets - Practicas con colecciones avanzadas
#### Este codigo tiene la finalidad de instruir en el uso de distintos tipos de colecciones.

* Pilas
* Colas
* HashSet
* Dicctionary
* List

Empleando objetos y tipos primitivos.
